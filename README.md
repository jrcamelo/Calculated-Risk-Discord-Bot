Calculated Risk is a bot that helps host image board Risk games on Discord.  
It makes tracking easier both for the game master and the players.  


The features are:
- Channel based games
- Turn management
- Rolls with message IDs
- Special roll highlights
- Alliances
- Leaderboards
- Levels
- ...and a lot more
